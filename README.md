# Student Grade Tracker

A GUI-based Student Grade Tracker developed using Java Swing.  
This project helps manage student records and calculate academic performance including average marks, highest marks, lowest marks, and grades.

Developed as part of the CodeAlpha Java Programming Internship.

---

## Features

- Add student names and marks
- Display student records in a table
- Calculate:
  - Average Marks
  - Highest Marks
  - Lowest Marks
- Automatic Grade Calculation
- Total Student Count
- Clear all student records
- Input validation
- GUI using Java Swing
