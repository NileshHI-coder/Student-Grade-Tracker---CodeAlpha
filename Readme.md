# Student Grade Tracker

A GUI-based Java application developed using **Java Swing** to manage and analyze student grades efficiently. This project was created as part of the Java Programming Internship Task at CodeAlpha.

---

## Features

- Add student names and marks
- Display student records in a table
- Calculate:
  - Average Marks
  - Highest Marks
  - Lowest Marks
- Automatic Grade Calculation (A/B/C/F)
- Total student count display
- Input validation for marks and empty fields
- Clear all student records
- User-friendly GUI using Java Swing

---

## Technologies Used

- Java
- Java Swing
- OOP Concepts
- ArrayList
- JTable
- Event Handling
